# JavaScript Logic Practice

This repository contains solutions to five beginner JavaScript logic exercises.

## Exercises

1. Find the Largest Number in an Array
2. Find the Sum of All Numbers in an Array
3. Reverse a String
4. Count the Number of Vowels in a String
5. Remove Duplicate Values from an Array

## Technologies Used

- JavaScript
- Git
- GitHub

## Project Structure

```
01-largest-number.js
02-array-sum.js
03-reverse-string.js
04-count-vowels.js
05-remove-duplicates.js
```

## How to Run

### Option 1: Using Node.js

1. Install Node.js from https://nodejs.org
2. Open a terminal in the project folder.
3. Run any JavaScript file using:

```bash
node 01-largest-number.js
```

You can also run:

```bash
node 02-array-sum.js
node 03-reverse-string.js
node 04-count-vowels.js
node 05-remove-duplicates.js
```

### Option 2: Using a Browser

1. Open Chrome or another browser.
2. Press **F12** to open Developer Tools.
3. Go to the **Console** tab.
4. Copy and paste the JavaScript code.
5. Press **Enter** to view the output.

### Option 3: Using an Online JavaScript Compiler

Copy and paste any JavaScript file into an online compiler such as:

- https://onecompiler.com/javascript
- https://www.programiz.com/javascript/online-compiler

Click **Run** to see the output.
